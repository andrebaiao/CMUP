## Setup

Create a venv:

```
$ python3 -m venv venv
$ source venv/bin/activate
```

Install the dependencies:

```
$ pip3 install -r requirements.txt
```

SetUp Database:

```
$ cd API    
$ python db.py
```

Run flask api on port 9000:

```
$ python3 -m app.py
```
